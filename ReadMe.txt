# Proyecto de Predicción de Cargas Eléctricas

Este proyecto tiene como objetivo desarrollar un modelo predictivo para la desagregación de cargas eléctricas en una central energética. Se utilizarán técnicas de aprendizaje automático para predecir el consumo de energía de diferentes equipos.

## Presentación del Proyecto

El modelo predictivo desarrollado en este proyecto será fundamental para optimizar el uso de la energía en la central, identificando patrones de consumo y proporcionando información valiosa para la toma de decisiones. El equipo trabajará en la recopilación y limpieza de datos, el diseño y entrenamiento del modelo, así como en la evaluación de su rendimiento.
